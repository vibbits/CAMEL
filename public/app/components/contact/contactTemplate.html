Contribute

We are looking forward to contributions from all experts throughout the world. To contribute with new entries, please fill out this template excel file with the metadata of your experiment and email it to us in attachment. Optionally, also attach variant data of genome-wide sequencing by using this template file.

For changes to existing entries, you only have to fill out the fields of metadata that you would like to change and the experiment's name. Adding mutation data can also be done in a similar way by using the same template file and mentioning the experiment's name.

For more complex changes or suggestions, please contact us directly. Contributors will always be acknowledged appropriately in future output.
